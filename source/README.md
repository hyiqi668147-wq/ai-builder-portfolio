# 黄一奇 | AI Builder Portfolio

这是黄一奇的 AI Builder 个人作品集首页，用于展示 ProofMe 项目的产品定义、AI workflow、构建过程和可验证工作证据。

## 核心项目

ProofMe 是一个 AI 求职能力证明生成器：输入岗位目标和真实经历，输出岗位能力证明、简历表达、面试回答和投递话术。

## 技术栈

- React
- Vite
- Tailwind CSS
- GSAP + ScrollTrigger
- Lenis
- lucide-react

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
